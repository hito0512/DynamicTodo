# DynamicTodo

思源笔记挂件 — 动态任务看板

## 功能

- **双视图切换**：看板视图 / 日历视图，标签切换
- **任务管理**：创建、编辑、删除任务
- **拖拽排序**：任务卡片拖拽到不同状态列
- **日期范围**：支持开始日期和结束日期，日历视图连续显示
- **任务预览**：鼠标悬停预览任务详情
- **Markdown 支持**：任务描述支持 Markdown 语法和超链接
- **状态自定义**：任务状态文本可自定义
- **数据持久化**：数据存放在思源块属性中
